# MikroTik Networking Lab

A virtual networking lab using MikroTik RouterOS in VirtualBox for practicing routing, firewall rules, and network configurations.

## Features

- MikroTik RouterOS VM setup
- SSH and WebUI access
- Nginx reverse proxy configuration
- VLAN and routing labs

## Requirements

- VirtualBox
- MikroTik RouterOS ISO
- Debian/Ubuntu host
